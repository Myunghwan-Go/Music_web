'Music Site'
'Music Site'를 통해서 나의 플레이리스트를 이동시켜보세요!
'Music Site'는 사이트 간 데이터 공유를 가능하게 함으로 그동안 사이트 간
음악 플레이리스트 공유가 불가능하여 불편함을 겪던 사용자들의 실제 필요에
대응하는 유용한 서비스입니다.
[플레이리스트 추출하기]
‘플레이리스트 추출하기’
기능은 플레이리스트 이동을
위하여 사용자가 소유하던
개인 플레이리스트 정보를
추출합니다.
[플레이리스트 업로드하기]
‘플레이리스트 업로드’
기능은 추출한 리스트를
원하는 음원 사이트의
플레이리스트에
업로드합니다.
[음악 추천 받기]
‘음악 추천 받기’
기능은 사용자가 선택하는
장르에 맞추어 사용자에게
최대 10개의 음악을
추천해드립니다.
사이트 사용 설명
플레이리스트 추출하기
1. 해당 웹 사이트 아이디 작성
2. 해당 웹 사이트 비밀번호 작성
3. 추출하기 버튼 클릭
플레이리스트 업로드하기
1. 해당 웹 사이트 아이디 작성
2. 해당 웹 사이트 비밀번호 작성
3. 플레이리스트 파일 선택
4. 올리기 버튼 클릭
음악 추천 받기
1. 장르 작성
2. 추천 받기 버튼 클릭
